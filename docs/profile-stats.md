# Profile Stats

The profile cards in `assets/stats/` are generated by
`scripts/generate-profile-stats.mjs` and updated by
`.github/workflows/profile-stats.yml`.

The workflow runs once per day at `07:23 UTC` and can also be triggered
manually from GitHub Actions.

## Required secret

Add an Actions secret named `PROFILE_STATS_TOKEN`.

Use a personal access token that can read the repositories you want included in
the profile statistics. Without this secret, the workflow fails intentionally so
it does not overwrite accurate private/token-visible stats with incomplete
public-only data.

## Generated cards

- `github-stats.svg`: stars, commits, PRs, issues, last-year contributions, and
  private/restricted contribution count.
- `top-langs.svg`: language distribution from visible owned repositories.
- `github-streak.svg`: current and longest contribution streaks.
- `contribution-graph.svg`: last-year contribution calendar.
